# ⚡ Máster en Fundamentos de JavaScript

Una guía de estudio y referencia técnica que recopila los pilares fundamentales del lenguaje JavaScript (ES6+). Este repositorio está estructurado de forma modular para servir como base sólida en el desarrollo de lógica de programación y aplicaciones web dinámicas.

---

## 🗺️ Ruta de Aprendizaje Organizada

El proyecto se divide en módulos progresivos para garantizar el dominio del lenguaje:

* **📁 01-Tipos:** Variables, constantes, tipos primitivos, tipado dinámico, estructuras de datos compuestas (Objetos y Arrays) y funciones esenciales.
* **📁 02-Operadores:** Operadores aritméticos, asignación, comparación estricta, lógica booleana, evaluación *Falsy*, operadores de bits (*Bitwise*) y el operador ternario.
* **📁 03-Control de Flujo:** Estructuras condicionales (`if`, `else`, `switch`) y bucles fundamentales (`while`, `do-while`, `for`, `for-of`, `for-in`), incluyendo el manejo de saltos con `break` y `continue`.
* **📁 04-Ejercicios:** Bloque de 10 desafíos prácticos enfocados en resolver problemas de lógica pura y algoritmos.
* **📁 05-Objetos (POO):** Profundización en programación orientada a objetos, abarcando la naturaleza dinámica de las propiedades, funciones de fábrica (*Factory Functions*), funciones constructoras, paso por referencia vs. valor y listado de propiedades.

---

## 🛠️ Stack Tecnológico
* **Lenguaje:** JavaScript Vanilla (ECMAScript 6+)
* **Entorno:** Navegador Web / Consola de Desarrollo (DevTools)
* **Estructura Base:** HTML5 (Inyección de Scripts dinámicos)
